# Murmur Dev Mono Logo Concepts

This pass refines the preferred top-left concept from the first mock board: an amber waveform forming a soft lowercase `m`, paired with a text cursor to hint at transcription and developer tooling.

The wordmark uses a monospaced font stack so it feels code-adjacent without becoming harsh:

`JetBrains Mono`, `SFMono-Regular`, `Roboto Mono`, `IBM Plex Mono`, `Menlo`, `Consolas`, `monospace`

Files:

- `murmur-logo-dev-mono-primary.svg`: horizontal lockup
- `murmur-app-icon-dev-mono.svg`: icon-only app mark
- `murmur-logo-dev-mono-stacked.svg`: stacked presentation lockup
